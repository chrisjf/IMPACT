Help

Here you can find instructions for using the IMPACT toolbox prototype and its tools. The IMPACT toolbox offers four different tools for argument reconstruction, visualization, consultation and policy modelling. The separate instructions for each tool will guide you through the main functions of the toolbox to let you experience how the tools work in the context of online policy consultations. Feel free to use other functions of the tools as well, but please note that the toolbox is still a prototype and some functions may not work as intended.

The IMPACT toolbox is optimized for the following browsers: Google Chrome, Mozilla Firefox
